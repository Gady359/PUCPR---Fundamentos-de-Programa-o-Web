Neste trabalho utilizei conhecimentos que adquiri durantes as semanas da PUC, e em conjunto dos cursos e formações na plataforma de ensino de programação DIO. Queria fazer a escolha de serviços como checkbox, porem com o método GET não permite o envio de múltiplos dados, sendo necessário o POST, como isso quebraria um dos requisitos do tra balho tornei como textbox.

    Requisitos do trabalho:

  1. Utilizar os elementos HTML para estruturar o conteúdo do documento web, mantendo regras de estilo e códigos de script em arquivos separados, organizados em subpastas por tipo (script, imagens e estilos).
  2. Utilizar regras de estilo CSS para formatar a apresentação visual dos elementos HTML, organizadas preferencialmente em classes que podem ser reaproveitadas em diferentes elementos HTML.
  3. Utilizar Javascript para prover dinamicidade à interação com o usuário, como por exemplo: para controle de menus ou janelas de pop-up, ou validação de dados, ou alteração de estilo de elementos HTML, entre outros.
  4. Criar formulário, com validação de TODOS os seus campos, e recuperar os dados enviados via método GET com Javascript.
  5. Utilizar o caminho relativo para fazer referência a recursos internos ao site.
